# LuminaDE v0.4 ZIG

LuminaDE v0.4 to stabilny milestone projektu oparty o Zig + River, domykający fundament sesji GUI-first oraz codzienny UX oparty o mysz i multi-monitor.

## Highlights

- GUI-first runtime dla `panel`, `launcher`, `settings`
- Dojrzały `luminade-sessiond`:
  - supervisor procesów
  - restart policy z backoff
  - IPC przez unix socket + fallback queue
  - snapshot stanu sesji
- Dojrzały UX myszki:
  - `click-to-launch`
  - menu kontekstowe (favorite add/remove, remove-history)
  - anti-flood launch guard
- Multi-monitor UX:
  - `widget-id@output`
  - per-output launch guard
  - przełączanie workspace per output (`focus-output` + `set-focused-tags`)
- Login/session packaging:
  - wpis sesji Wayland do DM
  - autostart `sessiond`
  - skrypt instalacyjny plików sesji
- Settings vNext:
  - i18n PL/EN dla etykiet i komunikatów GUI
  - sekcja `System & Power` (lock/suspend/logout + audio up/down/mute)

## Zmiany techniczne

- Nowe aplikacje i biblioteki Zig: `apps/*`, `libs/*`
- Runtime i konfiguracja: `build.zig`, `build.zig.zon`
- Session/login assets: `config/wayland-sessions`, `config/autostart`, `config/luminade-session*`
- Narzędzia operacyjne: `scripts/sessionctl.sh`, `scripts/install-session-files.sh`, workflow fork/sync River
- Dokumentacja i roadmapa:
  - architektura
  - spec UX
  - spec panel/launcher
  - pakietowanie Fedora/Arch
  - roadmapa v0.4 -> v0.5

## Kompatybilność

- Wymagany Zig: `0.13+`
- Docelowe środowisko: Wayland + River

## Znane ograniczenia

- Pełny natywny panel `layer-shell` pozostaje otwarty i zaplanowany na kolejne wydanie
- Część modułów ustawień (np. pełny moduł ekranów / keybind editor) będzie rozwijana etapowo

## Następny krok (v0.5)

- Domknięcie pełnego natywnego panelu `layer-shell`
- Rozbudowa aplikacji ustawień (ekrany, skróty, domyślne aplikacje)
- Szersze i18n w panelu i launcherze + fallback locale chain

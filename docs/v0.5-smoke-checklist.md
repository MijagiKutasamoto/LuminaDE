# LuminaDE v0.5 Smoke Checklist (PL)

Szybka checklista do potwierdzenia, ze panel/launcher/settings/dekoracje oraz ikony dzialaja poprawnie na hostowym Wayland.

## 1. Warunki startowe

1. Potwierdz sesje Wayland:

```bash
echo "$XDG_SESSION_TYPE"
echo "$WAYLAND_DISPLAY"
echo "$XDG_RUNTIME_DIR"
```

2. Potwierdz Zig:

```bash
zig version
```

## 2. Instalacja plikow sesji i ikon

1. Uruchom instalator:

```bash
sudo ./scripts/install-session-files.sh
```

2. Sprawdz, czy ikony sa zainstalowane:

```bash
ls -l /usr/share/icons/hicolor/scalable/apps/luminade.svg
ls -l /usr/share/icons/hicolor/scalable/apps/luminade-settings.svg
ls -l /usr/share/icons/hicolor/symbolic/apps/network-wireless-symbolic.svg
ls -l /usr/share/icons/hicolor/symbolic/apps/audio-volume-high-symbolic.svg
```

3. Sprawdz wpisy `Icon=` w desktop entry:

```bash
grep -n '^Icon=' config/wayland-session.desktop config/wayland-sessions/luminade.desktop config/autostart/luminade-sessiond.desktop
```

## 3. Build i uruchomienie komponentow

1. Build:

```bash
zig build
```

2. Panel (daemon):

```bash
zig build run-panel -- --daemon
```

3. Launcher (daemon):

```bash
zig build run-launcher -- --daemon
```

4. Settings:

```bash
zig build run-settings
```

## 4. Test mockup parity (manualny)

1. Panel:
- badge trybu jest wycentrowany,
- workspace + app pills sa czytelne i klikalne,
- statusy `Net/Vol/Bat` sa widoczne.

2. Launcher:
- centralna karta overlay,
- pole wyszukiwania i lista wynikow maja rowny pionowy rytm,
- ikony przy wpisach sa obecne.

3. Dekoracje:
- launcher ma titlebar i zaokraglenia,
- settings ma titlebar i zaokraglenia,
- panel nie ma dekoracji okna.

## 5. Test event queue (szybki)

1. Panel click test:

```bash
cat > .luminade/gui-panel-events.tsv <<'EOF'
click	app-terminal@eDP-1
click	sys-audio@eDP-1
click	ws-2@eDP-1
EOF
```

2. Launcher click test:

```bash
echo "firefox" > .luminade/gui-launcher-query.txt
cat > .luminade/gui-launcher-events.tsv <<'EOF'
click	result-0
EOF
```

3. Settings click test:

```bash
cat > .luminade/gui-settings-events.tsv <<'EOF'
window-mode
display-layout
default-terminal
system-network
audio-volume-up
EOF
```

## 6. Kryteria zaliczenia

- Brak bledow parsera/kompilacji dla `panel`, `launcher`, `settings`.
- Ikony desktop + symbolic faktycznie widoczne po instalacji.
- Panel/launcher/dekoracje wizualnie zgodne z kierunkiem mockupow `mockupD.png` i `mockupL.png`.
- Event queue dziala bez crashy i bez floodowania logow.
